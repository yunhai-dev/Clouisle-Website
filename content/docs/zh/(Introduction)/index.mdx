### 什么是 Clouisle（云屿）？

云屿是一个企业级 AI 平台，将 AI Agent、工作流编排与知识检索整合到同一个生产级系统中。

### 核心能力

- **AI Agent** — 构建和部署支持多模型的智能 Agent
- **工作流编排** — 通过可视化工具设计复杂的自动化流程
- **知识库** — 企业级 RAG，同时支持传统检索与 Agent 驱动检索
- **工具集成** — 通过 MCP 协议连接外部工具和 API
- **安全与治理** — SSO、RBAC、审计日志与私有化部署
