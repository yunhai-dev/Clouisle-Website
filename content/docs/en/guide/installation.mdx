### Prerequisites

- Docker 24+ and Docker Compose v2
- At least 4 GB RAM available
- PostgreSQL 15+ (or use the bundled container)
- One or more LLM API keys (OpenAI, Anthropic, etc.)

### Quick Start with Docker Compose

```bash
# Clone the repository
git clone https://github.com/yunhai-dev/Clouisle.git
cd Clouisle

# Copy environment template
cp .env.example .env

# Start all services
docker compose up -d
```
