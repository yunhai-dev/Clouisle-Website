## 安装

> 内容即将补充。

### 前置条件

- Docker 24+ 和 Docker Compose v2
- 至少 4 GB 可用内存
- PostgreSQL 15+（或使用内置容器）
- 一个或多个 LLM API Key（OpenAI、Anthropic 等）

### 使用 Docker Compose 快速启动

```bash
# 克隆仓库
git clone https://github.com/yunhai-dev/Clouisle.git
cd Clouisle

# 复制环境变量模板
cp .env.example .env

# 启动所有服务
docker compose up -d
```
