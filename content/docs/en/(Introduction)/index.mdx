### What is Clouisle?

Clouisle is an enterprise AI platform that combines AI agents, workflow orchestration, and knowledge retrieval into one production-ready system.

### Core Capabilities

- **AI Agents** — Build and deploy intelligent agents with multi-model support
- **Workflow Orchestration** — Design complex automation pipelines with visual tools
- **Knowledge Base** — Enterprise-grade RAG with both traditional and agentic retrieval
- **Tool Integration** — Connect external tools and APIs via MCP protocol
- **Security & Governance** — SSO, RBAC, audit logs, and private deployment
